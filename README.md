# Task 1 - Sales & Revenue Analysis Dashboard

## Objective
Build an interactive dashboard to analyze sales and revenue data.

## Tools Used
- Power BI
- Excel

## Features
Total Sales KPI
Total Orders KPI
Total Quantity KPI
Total Profit KPI
Monthly Sales Trend
Monthly Profit Trend
Top Products by Sales
Region Filter
Category Filter

## Insights
Total Sales: 339M
Total Profit: 56.34M
Electronics category generated highest sales.
Laptop was the top-performing product.
Regional analysis helps identify high-performing areas.
